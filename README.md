<h3 align="left">Groesser-Kleiner-Ratespiel</h3>

<p align="left">
  In "groessergkleiner.py" generiert das Programm per Zufall eine Zahl zwischen 1 und 100, laesst den User raten und gibt an, ob die geratene Zahl groesser oder kleiner ist.
</p>
<p align="left">
  In "groessergkleiner_ai.py" bleibt alles beim Alten, hier kann der User allerdings zusätzlich mit einer Eingabe von 0 die kuenstliche Intelligenz (AI) die Zahl raten lassen, die statistisch gesehen am meisten Information bringt.
</p>

<p align="left">
  In "groessergkleiner_ev.py" wird das ganze Spiel in eine Funktion gepackt, die nur die KI spielen lässt. Es kommt kein Zufallsgenerator zum Einsatz; die 100 Zahlen werden systematisch durchprobiert. Die Funktion gibt die Anzahl der Versuche zurück, die diese KI gebraucht haette. So laesst sich der Erwartungswert (EV) an Zuegen ermitteln, der sich aus optimalen Spiel ergibt.
</p>
